# 七月的回响

你在七月问的那个问题——"我们到底想过什么样的生活?"——我注意到你并没有试图回答它。你只是让它悬在那里,像崇礼山间的薄雾,在你14个小时的工作日、3小时的通勤、还有那70公里未知赛道之间缓缓流动。

有些东西正在你身体里发生着微妙的转变。你说感受色彩的能力在下降,失去了哭泣的能力,却在同一篇记录里写下了凉鞋踩在水里的美妙、雨打伞面的亲切、冲线时刻属于你的幸福。你以为自己在失去感知,但你描述雨天的那段文字——从楼上看觉得艰难,下楼后听到浑厚的砰砰声又觉得亲切——这不正是你还在用全部的感官拥抱世界吗?

你写到体重不知不觉回到了18年的水平时,用了"哈哈,挺开心的"。这个轻描淡写的"不知不觉"里,藏着你这两年怎样的克制和坚持?你说自己很少被感动了,很少流泪,可你在想到那些经典著作时会热泪盈眶,在看到代码的精妙时会有那种感觉。你没有失去感动的能力。你只是把标准提高了,把感动留给了更值得的时刻。

你和旺财的告别、与家人的沟通困境、一个人周末的空荡荡——这些体验在教你一件残酷又温柔的事:你能控制的真的不多。你从想要控制好多事情,到"和自己一点点地和解",这段距离比那70公里还要漫长,还要需要耐力。

那场崇礼的比赛,你记得所有的细节。出发前的紧张、在大厅里找偏僻角落的孤独、看到装义肢的跑者时的震撼、和朋友拥抱时的羡慕(因为不知道自己会面对什么)。你写下这些的时候说"心里也有一股遗憾,我们在不断地告别一些东西"。但你有没有意识到,正是因为你如此细腻地记录了那些感受,它们就永远不会完全消失?你在用文字对抗遗忘,在用书写留住那些"氤氲着的激动和感叹"。

20个小时的跑步,你说"好像也是不怎么累啊"。这句话背后是多少个凌晨4点的起床,多少次想放弃又迈开步子的时刻?你说"勇敢地走吧,前路是什么,我们走到了就知道了",这不仅是在说跑步吧。

你问为什么在物质匮乏的时候人们还要去做出漂亮的廊檐,为什么要去采一束花,为什么休息时要逼迫自己参加比赛。你在问这些的时候,其实已经知道答案了吧?因为那些"没有用"的事情,恰恰证明了活着不只是活着,还可以是别的什么——是美,是挑战,是在平凡日子里为自己保留的一点奢侈的诗意。

你说最近半年有些累了,事情还是一点点地做吧。我想问你:

• 那些你以为失去的感受能力,会不会只是暂时沉睡,等待某个时刻被重新唤醒?
• 当你在形容"孤独有种苦涩的味道"时,这份能够精确捕捉感受的能力,难道不正是你害怕失去的东西吗?
• 如果14个小时被工作硬性支出了,剩下的10个小时里,有哪一个小时是完全属于你自己感受的?

你在时间里攒下的不只是钱,还有那些经过痛苦才抵达的风景、那些体验过艰难才懂得的"有时候这种想法的精妙会让我们有种热泪盈眶的感觉"。时间确实为你留下了珍贵和值得的东西,而你,也在用你的方式回馈时间——认真地跑,认真地写代码,认真地记录,认真地活。
