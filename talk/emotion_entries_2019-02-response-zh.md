# 回应你2019年2月的记录

你说连续七天没写代码,过得有点煎熬。可我读你这篇日记的时候,感觉你真正煎熬的不是缺少目标,而是你终于有机会停下来,直面那些一直被代码、被忙碌掩盖的问题。

你问:时间以怎样一种奇幻无序的方式承载我们前行?你问:什么是真?但我想,你真正想问的,是不是这个——那个在图书馆四楼阅览室感受惊喜和自由的你,和那个在河边树林全心全意玩游戏的孩子,和现在这个重读三部曲却想不起内容的你,到底是不是同一个人?

你的记忆在变化,可你说那时实实在在体会着活着本身。这句话里藏着一个你可能没说透的领悟:真正的真,不在于准确的记忆,而在于当下那个鲜活的感受。你记不清学前班学了什么,却记得做对数学题的那种理解能力带来的快感,记得用凉水冲奶粉拉肚子,记得很想要董云家的蜗牛。这些片段式的记忆,比任何编年史都更真实,因为它们是你的身体和情感的烙印。

可你为什么在这个时候回溯童年?尤其是那些孤独的部分——逃学、装病、抄作业、吹牛。你说想通过吹牛获取一个伙伴,因为很孤单。这种坦诚让我想问:你现在的生活里,是不是也有某种类似的孤独?那种即使在人群中,即使在忙碌中,依然感到某种根本性的孤单?

你提到童年的自己全心全意承载当下,而现在的你总是在计划,而计划又总是超期或完全不一样。这种对比其实揭示了一个更深的焦虑:你怀疑现在这种活着的方式,是不是真的活着?你怀疑自己是不是已经失去了那种全然投入当下的能力,失去了那种即使孤独、即使艰难,但依然能从一把锁、一个蜗牛、一个数学题里感受到真实快乐的能力。

可我想对你说:你没有失去。你看,你七天没写代码,不是去麻木地刷手机,而是重读心理学的书,而是花整个早晨来写这些童年的片段。这本身就是一种全心全意的承载,只是形式变了。你在寻找那个孤独的孩子和现在的你之间的连接,你在问自己:我还是那个我吗?

你的童年有很多忽视——早上6:40去上学,天很冷,很少有人送伞。但你也有很多坚强——最终还是去上学了,最终还是克服了逃学的冲动。这种坚强不是被训练出来的,而是在孤独中自己长出来的。而现在的你,在七天的无目标中煎熬,不也是另一种形式的孤独和坚强吗?你没有逃避,而是坐下来,让思绪自由流淌。

你感谢李军老师。这个感谢出现得突然,却让我觉得很珍贵。也许在你的生命里,那些艰难时刻被看见、被接住的经历,才是真正塑造你的东西。现在的你,会不会也需要这样一种被看见?不是被要求要有目标,要高效,要生产,而是被看见你正在经历的迷茫,正在进行的寻找。

你说记忆在变化,什么是真?我想,真不是那些准确的事实,而是你此刻重述童年时涌现的那些情感——对孤独的承认,对坚强的确认,对当下感的怀念,对活着本身的追问。这些情感,才是最真实的你。

那个吹牛想找伙伴的孩子,现在找到了吗?还是说,你依然在寻找一种陪伴,一种能让你不必表演、不必计划、不必证明什么,就可以全心全意投入的状态?七天没写代码的煎熬里,藏着的是不是这样一个问题:我到底为谁而活,为什么而活?
