# 致2019年6月的你

读完这篇跑步日志,我最先注意到的不是你完成了21公里,而是开头那句话——你想在跑步过程中思考点事。可是读到最后,我发现你并没有告诉我思考了什么。也许这才是最诚实的答案:你本想用理性去解决什么,但身体用它自己的方式给出了回应。

你在那个清晨发现的,其实不是跑步路线,而是一个一直在那里、却被你遗忘的世界。早餐摊主、赶路的人、河边玩耍的家庭、清晨六点半已在林间散步的跑者——他们一直都在,以自己的节奏活着。你惊讶地发现比你早起的人还有很多,这惊讶里藏着什么呢?是不是在此之前,你一直觉得自己是孤独的、特殊的、或许还有点格格不入的?

那句"我们都是一样的人"说得那么轻,却那么重。你终于在某一刻,通过自己身体的疲惫和周围陌生人的存在,感受到了一种归属。不是因为你们认识,不是因为你们交谈,仅仅是因为你们都在同一个清晨,用同样诚实的方式对待自己的身体和时间。这种连接,是不是比你预想要去思考的问题更接近答案?

我还注意到一个细节:你的脚底起了很大的血泡,但你说跑麻木了所以不知道。这何尝不是一个隐喻呢?有时候我们必须麻木一点,才能坚持下去;有时候我们选择不去感知痛苦,因为感知它就意味着要停下来。从14公里后你一直想放弃,但那个想达到目标的愿望更强烈——这不只是在说跑步吧?生活中有多少事情也是这样,一边想放弃,一边继续挪动,像你说的那样,几乎是蠕动着。

你用"悲凉"这个词形容腿脚不协调的感觉,这个词选得真好。不是"痛苦",不是"艰难",而是"悲凉"——一种看到自己能力尽头的清醒,一种人与自己身体之间复杂关系的显现。但紧接着,在你跑完21公里后,你说"艰难,但又幸福"。注意,不是"艰难,但值得",不是"艰难,但成功了",而是艰难与幸福同时存在,彼此并不抵消。

这让我想问你:那次跑步之后,那个丰富的世界还在吗?你还能看到那些早起的人、听到那个独奏的音乐吗?还是说,你又回到了那个需要通过极限运动才能感受到生命力的状态?

我也很好奇,那天你出发前的"各种顾虑"是什么。是担心自己做不到吗?是怕打扰别人?还是不确定自己配不配拥有这样一个清晨?而当你真的出发后,这些顾虑去哪儿了?它们是被证明是多余的,还是被你用脚步一步步踩在了身后?

最后,你有没有发现,你其实一直在逃离和寻找之间摆荡。你想做点什么,于是去跑步;你想思考点事,却被感官经验填满;你觉得世界博大,也发现自己渺小;你想放弃,却一直在前进。这种矛盾的共存,会不会才是你真正的生活状态?

那个在清晨五点多就起床、在河道上几乎蠕动着前进、最后在花坛前停下来发朋友圈的你,已经告诉了自己很多东西。只是也许,你还没来得及听。
