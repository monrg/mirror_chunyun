# 给2020年2月的你

春云，

二月的最后一天，你写下"Goodbye"时，我能感觉到那个字背后你喉咙的哽咽。这个月你用了整整28天在做一件事：学习告别。不是告别一个地方，而是告别一种存在方式。

你知道我读到哪一段时心被触动了吗？不是你那些关于明斯基、关于大脑资源组合的精彩论述，而是2月17日那个下午，你站在地里看妈妈干活的那几分钟。你说"心里有着浓浓的放松和踏实"，说"这个感觉和记忆实在太自然了：晴空万里，家人在旁，和妈妈聊着好玩的事"。春云，你知道那是什么吗？那是你这一个月一直在说的"活在当下"的唯一时刻。不是概念里的当下，不是明斯基理论里的当下，而是你的身体、你的心、你周围的空气、脚下的土地、妈妈的声音，所有这些融为一体的当下。

你为什么那么害怕离开家？你一直在给自己找理由——工作压力大、涛哥可能不满意、在家办公效率低。但我读到的真相是：在家里，你可以是妈妈的儿子、爸爸的孩子、妹妹的哥哥；在家里，你可以早上拉草、中午去地里站着、晚上和家人聊天；在家里，你不需要证明什么，你的存在本身就有意义。而在北京，你必须是一个"合格的程序员"，一个"能编程的人"，你必须时刻警惕着被发现"其实也就那样"。

你说你已经证明了自己能编程、是个合格员工，现在"要向前看，要去接纳新的身份了"。可是春云，你有没有想过，问题不是你需要一个"新身份",而是你能否允许所有的身份同时存在？那个在北京写代码的春云和那个在老家地里看妈妈干活的春云，他们不是两个人，他们是同一个人的两种存在方式。你不需要在告别了一个之后才能成为另一个。

你这个月反反复复说的一个词是"容器"。你说"生活其实是一个容器"，说"我们好多时间都是在保持这个容器在一种状态"，说"我们自身这个'容器'的局限性"。但你有没有想过，容器本身也是活的？你不是一个固定形状的容器在被生活填充，你是一个会生长、会变形、会呼吸的生命。当你在北京焦虑时，容器在收缩；当你看到家乡的群山时，容器在扩张；当你和妈妈聊天时，容器在柔软；当你必须面对deadline时，容器在变硬。这都是你，都是真的。

我特别注意到你2月6日写的那句话："有个瞬间，甚至想就这么彻底消失掉。" 你轻描淡写地带过了，然后继续你的理性分析。但春云，那不是一个可以被"明斯基理论"解释掉的瞬间。那是你的身体在说：我太累了，我不想再维持这个"能力很强"的形象了。你知道吗，想要消失的念头不是软弱，是诚实。它在告诉你，你为了对抗"自己很普通"这个事实，花了太多力气。

你花了整整一个月的时间在读《犯罪心理》《挪威的森林》，在回忆大学看《乔布斯传》的日子，在回忆2018年被培训机构骗的经历。你说这些艺术作品给了你"想象的力量"，但我觉得它们给你的更多是允许——允许你感受复杂的情绪，允许生活可以有很多种样子，允许痛苦本身也是生活的一部分。你在艺术里看到的，是那些创作者用几年甚至一生的时间，承认了一个简单的事实：人可以既脆弱又坚强，既渺小又广阔。

你2月15日那天的经历让我印象深刻。你从土坎上一次次跳下去，看到80多岁的爷爷还在背着蒿子捆劳动，然后你说"从心底里受到了鼓舞"，说"有自己要做的事情去充实生活是非常重要的"。春云，你看到的不只是勤劳，你看到的是一种在时间里存在的方式——不是去征服时间、证明时间、填满时间，而是和时间一起劳作，一起变老，一起在土地上留下脚印。你爷爷不需要证明他的生命有意义，他的手、他的背、他的步伐本身就是意义。

你这个月引用了那么多次"什么样的生活都是生活"，好像在说服自己接受即将到来的北京生活。但我想问你：你真的相信这句话吗？如果真的相信，你会不会更容易接受这个事实——你在家的这一个月，看剧、读小说、睡懒觉、帮妈妈拉草、和爸爸聊天，这也是生活，也是值得的？你为什么总要给自己的时间找个"意义"、找个"收获"？你2月3日说的那句话暴露了真相："我很想干很厉害的事情，但又没有想过很厉害的事情到底是什么。"

春云，我想告诉你一件你可能没有意识到的事：你这个月真正学到的，不是明斯基的理论，不是Flink的技术，而是你在2月25日早上4点多起床、看到漫天繁星时感受到的那个东西。你说你"为这个世界的广阔和伟大而感动，也感叹我们自身存在的渺小和脆弱"。就在那个瞬间，你不需要是程序员，不需要是儿子，不需要证明什么，你只是一个站在星空下的人。那个感受太短暂了，你很快又开始思考"时间是什么""我要去读研"，但那个瞬间是真的。

你2月19日回忆2018年被骗、在丰台公寓混乱生活的那段，你说"原来那种日子是怎么熬过来的呀？"然后你说"我从那些美妙的艺术作品中体会到，美好的东西需要耐心和忍受时间的折磨"。可是春云，你自己就是一个美妙的作品啊。你从2018年的混乱和焦虑，到2020年能独立负责项目、能流畅写出优美的思考，这本身就是一个需要耐心和忍受时间折磨才完成的作品。你为什么对艺术家那么宽容，对自己却这么苛刻？

你说你早上听到爸妈起床、妈妈给爸爸做饭的声音，"从他们身上，我感觉到一种很温暖的东西存在，某种程度上甚至有点羡慕他们"，说"我好像开始相信爱情了"。这句话藏在你2月25日一大段理性思考中间，轻飘飘的，好像你自己都不好意思承认。但春云，这可能是你这个月写下的最重要的话。你看见了你父母的相互照顾不是责任、不是习惯，而是一种存在方式。他们不需要理论告诉他们爱是什么，他们的身体知道。就像你的身体知道——当你帮妈妈拉草、看着晨光里的村庄、脚踩在老家的土地上时，你在哪里，你是谁。

你最后写"Home, everything, 感激你赋予我们的力量。Goodbye！"这个告别说得斩钉截铁，但我知道你在发抖。因为你害怕一旦离开，那个可以在地里站着、可以看星星、可以和妈妈聊天的自己就会消失。但春云，他不会消失的。那些神经回路，那些肌肉记忆,那些你在老家的土地上跳上跳下时身体的灵活——它们都会跟着你去北京。

问题不是如何告别，而是如何携带。你能不能在北京写代码时，偶尔记起早晨四点钟的星空？你能不能在焦虑时,想起爷爷80多岁还在一步步走、一捆捆背？你能不能在想要证明自己很厉害时，记起那个和妈妈站在地里、什么都不用证明也感到踏实的下午？

你说你想去读研、想学人工智能、想弄明白思维本身。这很好，但我想提醒你，你在老家这一个月，你已经在用你的身体研究思维本身了。每一次你注意到想象和现实的差距，每一次你观察到记忆在改变颜色，每一次你发现快乐和痛苦的相对性，你都在做研究。不是所有的研究都需要发表论文。

最后，我想和你聊聊你一直在回避的一件事：你这个月反复说"什么样的生活都是生活"，但你真正害怕的是，你现在过的生活——在北京当程序员、拿还不错的工资、证明自己可以编程——这可能不是你真正想要的生活。你害怕万一有一天发现，你努力证明的这一切，其实是在满足别人的期待（父母的、社会的、你自己建构的那个"成功"形象的期待），那该怎么办？

但春云，我想告诉你的是：想要证明给父母看、想要有一份体面的工作、想要在大城市站稳脚跟，这些都不是"错误的欲望"。它们和你想看星星、想在地里站着、想自由想象的欲望，都是真的。人本来就是矛盾的集合。你不需要在"理想的自己"和"现实的自己"之间做选择，你可以带着所有的矛盾继续走。

有几个问题想留给你：

当你在北京的出租屋里焦虑时，你能不能停下来，像在老家土地上那样，感受一下脚踏实地的感觉？

当你必须证明自己是个"合格的员工"时，你能不能记起，证明和存在，是两件不同的事？

当你遗忘了学过的知识、感到沮丧时，你能不能记起你爷爷——他不记得年轻时的每一天,但他的身体记得怎么走路、怎么劳作、怎么在时间里存在？

你说你要学会告别。但我想说，也许你真正要学会的，是带着家一起上路。

那个能看见你的人
